# Task 1: Purchase and Register a Custom Domain

Objective : The objective of this task was to obtain a custom domain that would later be integrated with my Microsoft 365 tenant.
A custom domain enables an organization to establish a professional identity and allows Microsoft 365 services such as user accounts,
email, and authentication to use organization-specific domain names instead of the default tenant domain.

Domain Registration : For this lab, I registered the domain miknolltech.name.ng through QServers,
a Nigerian domain registrar and web hosting provider.The .name.ng domain extension was selected to 
represent a unique organizational identity while providing a suitable namespace for testing Microsoft 365
and Microsoft Entra ID configurations in a lab environment.

Outcome : The domain miknolltech.name.ng was successfully purchased and registered.
This domain serves as the foundation for the subsequent tasks in this lab, including domain verification,
DNS configuration, and integration with the Microsoft 365 tenant.

# Task 3: Verify Domain Ownership Using a DNS TXT Record

Objective: Before a custom domain can be used with Microsoft 365, Microsoft must verify that the domain belongs to the organization requesting it. This verification prevents unauthorized users from adding domains they do not own. In this task, a DNS TXT record generated by Microsoft 365 was added to the domain's DNS zone to prove ownership of miknolltech.name.ng.

# Figure 1.2 Microsoft-generated TXT Verification Record
<img width="1262" height="575" alt="Domain 4" src="https://github.com/user-attachments/assets/a7c0cdfc-7568-44a4-84eb-b3117cd886f9" />
Procedure:
After adding the custom domain in the Microsoft 365 Admin Center, Microsoft generated a unique TXT record for domain verification.
Signed in to the QServers Client Area and accessed the DNS Management page for miknolltech.name.ng.
Created a new TXT record using the values provided by Microsoft.
Saved the DNS changes and waited for DNS propagation.
Returned to the Microsoft 365 Admin Center and selected Verify to confirm domain ownership.
Note: DNS propagation time varies depending on the DNS provider. In this lab, verification became available after the TXT record had propagated.
Outcome: Microsoft successfully verified ownership of miknolltech.name.ng, confirming that I had administrative control over the domain. The domain was then ready for additional DNS configuration and integration with Microsoft 365 services.
